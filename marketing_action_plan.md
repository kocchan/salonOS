**1日のルーティン**

- 21:00 〜 21:30：美容師アカウントの投稿にリプライ（共感ベース、宣伝なし）

**返信率を上げるコツ**

#### B. Googleマップ×コールドアプローチ

**電話番号収集の手順**

Googleマップで以下のエリア×キーワードで順番に検索し、各エリア20〜30件の電話番号をスプレッドシートに収集する。

対象エリア優先順位：
1. 東京（銀座・渋谷・表参道・中目黒・下北沢・吉祥寺）
2. 大阪（心斎橋・梅田・堀江）
3. 名古屋（栄・大須）

検索キーワード：「美容室」「ヘアサロン」「個人サロン」

収集する情報：店名、電話番号、口コミ件数（少ない＝個人経営の可能性大）、Webサイト有無。口コミ件数30件以下かつWebサイトがないかシンプルなものは優先ターゲット。

**電話スクリプト（30秒バージョン）**

```
「お忙しいところ失礼します。SalonOSというサービスを
```

4. 会えた場合はiPad/ノートPCでデモを3分だけ見せる（「予約が入ったらLINEで通知が来る」画面が最も刺さる）
5. 翌日LINEかメールでフォローアップ

#### C. Instagram美容師アカウントへのアプローチ

**ターゲットアカウントの見つけ方**

以下のハッシュタグを検索し、「最近の投稿」タブで直近2週間以内の投稿者を探す。

```
#美容師
#美容室オーナー
#サロンオーナー
#一人美容室
#フリーランス美容師
#美容室経営
#個人サロン
#ヘアサロン経営
#美容師の働き方
#サロン集客
```

選定基準：
- フォロワー数 500〜10,000（プロ意識があり、大手ではない）
- 投稿頻度 週2回以上（アクティブユーザー）
- 投稿内容にスタイル写真以外に「経営」「集客」「予約」「苦労」を含む
- Instagramのプロアカウントまたはプロフィールに「オーナー」の記載あり

**コメント → フォロー → DMの3ステップ**

Step 1（コメント）：投稿を見て、施術や内装への具体的な感想を1〜2行コメントする。「素敵なカラーですね！」ではなく「ハイライトの入れ方が絶妙で、光の当たり方で色が変わるの凄いですね」のように具体的に。宣伝は一切なし。

Step 2（フォロー）：コメントの翌日にフォローする。フォロバが来なくても構わない。

Step 3（DM、フォローから3〜5日後）：

```
○○さん、先日の投稿にコメントした者です。
美容師さんの働き方への想いが伝わってきて
フォローさせていただきました。

実は美容室向けのSaaS「SalonOS」を開発していまして、
HOT PEPPERに依存しない予約・カルテ管理を

もし集客コスト・予約管理で悩まれていたら
話を聞いてもらえますか？
30分のZoomデモを無料でやっています。
```

1日のアクション上限：コメント5件、フォロー3件、DM3件。過剰なアクションはスパム扱いされる。

- 紹介1件成立ごとに翌月の月額料金から1,000円割引（積み上げ型）
- 紹介3件で1ヶ月分の月額完全無料
- 紹介5件以上の「アンバサダー」認定：ロゴ掲載、専用Slackサポート、機能リクエスト優先対応

**紹介された美容師（被紹介者）へのメリット**
- 通常30件/月の無料枠を、紹介コード入力で90日間 Basicプラン相当（無制限）を無料で使える
- 初月キャッシュバック：Basicプランに移行後、初月2,980円を全額返金

**LTVから見た適切な紹介報酬の試算**

```
想定LTV計算:
  平均継続月数: 18ヶ月（SaaS業界中央値）
  平均ARPU: ¥3,500（Free→Basic移行率を加味）
  LTV = 3,500 × 18 = ¥63,000

CAC上限（LTVの1/3以下が健全）: ¥21,000

紹介報酬の設計:
  紹介者への月額割引 ¥1,000×平均24ヶ月 = ¥24,000相当
  → LTVの38%（高いが、CACゼロのオーガニック獲得なので許容範囲）

  被紹介者へのBasic90日間無料 = ¥8,940相当

  合計コスト/1件 = 約¥33,000
  → 有料転換後のLTV ¥63,000に対して十分にペイする
```

### ユニットエコノミクス

**CAC目標（オーガニック）**

```
オーガニックCAC =
  月の時間コスト（創業者の時給換算）÷ 月の新規有料顧客数

  時給換算: 3,000円/時 × 40時間/月 = 120,000円
  Month 1新規顧客: 3名
  Month 1 CAC = 120,000 ÷ 3 = ¥40,000

  Month 3新規顧客: 10名（累計20名）
  Month 3 CAC = 120,000 ÷ 10 = ¥12,000
  → 目標: ¥15,000以下（Month 3）
```

**CAC目標（SNS広告、Month 4以降に検討）**

```
Meta広告での想定CPC: ¥80〜150
LP転換率: 3%
無料登録→有料転換率: 25%

CAC = (100クリック × ¥100) ÷ (100 × 3% × 25%)
    = ¥10,000 ÷ 0.75 = ¥13,333

目標CAC広告: ¥10,000以下（LTVの1/5以下）
```

**Payback Period（回収期間）**

```
LTV: ¥63,000
CAC目標: ¥15,000（Month 3以降）

Payback Period = CAC ÷ ARPU
  = ¥15,000 ÷ ¥3,500 = 4.3ヶ月

目標: 6ヶ月以内での回収
（SaaS業界健全水準: 12ヶ月以内）
```
